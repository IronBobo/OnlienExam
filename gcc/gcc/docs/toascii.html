NAME

toascii - translate integer to a 7-bit ASCII character

 SYNOPSIS



#include <ctype.h>

int toascii(int c);

 DESCRIPTION

The toascii() function converts its argument into a 7-bit ASCII character.

 RETURN VALUE

The toascii() function returns the value (c & 0x7f).

 ERRORS

No errors are returned.
