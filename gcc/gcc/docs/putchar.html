NAME

putchar - put byte on stdout stream

 SYNOPSIS



#include <stdio.h>

int putchar(int c);

 DESCRIPTION

The function call putchar(c) is equivalent to putc(c, stdout).

 RETURN VALUE

Refer to fputc().

 ERRORS

Refer to fputc().
